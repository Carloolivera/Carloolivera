## Sobre mí

Soy desarrollador fullstack enfocado en construir productos digitales reales — no demos. Co-fundé **AIDO Digital Agency** con el objetivo de llevar software de calidad a negocios que todavía no lo tienen acceso.

Mi flujo de trabajo está basado en **vibecoding**: desarrollo iterativo asistido por IA, donde la velocidad y la calidad no están en conflicto. Uso Claude Code como copiloto principal para mantener el ritmo sin perder el criterio.

## Proyectos activos

### 🗒️ [aido-todo](https://github.com/Carloolivera/aido-todo)
Todo app completa construida con **Next.js 16 + React 19 + Prisma 7 + shadcn/ui**.
Dashboard con gráficos (DonutChart + BarChart), autenticación JWT, dark mode, sorting, toasts y 40 tests. MVP funcional con deploy-ready en Vercel.

Sistema de menú digital y pedidos para restaurantes. Actualmente en producción para **Lo de Mau** (pizzería, Chascomús).

Web corporativa de AIDO — multi-idioma (ES/EN/PT), animaciones con GSAP y Three.js, integración Mercado Pago.
